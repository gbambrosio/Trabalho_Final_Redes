# Trabalho_Final_Redes
Entrega 12/11/2025

Este projeto consiste no desenvolvimento de uma plataforma digital interativa, criada com HTML, CSS e JavaScript, com o objetivo central de promover a conscientização sobre o Novembro Azul — o Mês Mundial de Combate ao Câncer de Próstata. A página visa quebrar o tabu sobre o rastreamento da doença, enfatizando a importância da prevenção e do diagnóstico precoce. A solução oferece uma abordagem completa ao tema, incluindo informações sobre: Estatísticas e Visualização de Dados | Saúde e Risco | Tratamento | Usabilidade e Acessibilidade | Conformidade com a LGPD
